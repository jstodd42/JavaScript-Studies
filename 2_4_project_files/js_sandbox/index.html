2 Types of Data Types: Primitve and Reference Types

Primitive Type

Primitive types are information stored directly in the location that the variable accessess. This info is stored on the stack. And when you accress Primitive data, you access it by it's value.

  • String: Sequence of characters, letters, numbers, and symbols.
  • Number: Integers, decimals, floats in JavaScript
  • Boolean: Means True and False
  • Null: Intintional empty value
  • Undefined: Variable that has been assigned a value. All variables are undefined as default. You will get an error if you don't define a constant.
  • Symbol: From es6 const sym = Symbol();

Reference Type
